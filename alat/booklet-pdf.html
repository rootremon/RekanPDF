Cara Cetak Booklet yang Benar
1

Buka dialog cetak

Klik tombol "Cetak Langsung" atau buka file PDF unduhan di Adobe Acrobat / aplikasi PDF lainnya, lalu pilih Print.

2

Aktifkan cetak bolak-balik (Duplex)

Pilih Duplex / Two-sided → Flip on Short Edge (bukan Long Edge). Ini kunci agar halaman belakang tidak terbalik.

3

Atur ukuran kertas

Pastikan ukuran kertas di printer sesuai dengan opsi yang dipilih (A4, A5, dll). Hilangkan tanda centang "Fit to page" / "Shrink to fit".

4

Susun, lipat, dan jilid

Setelah dicetak, susun semua lembar bertumpuk sesuai urutan (Lembar 1 paling atas), lipat di tengah sekaligus, lalu stapler di bagian lipatan.

Cara Membuat Booklet

1

Upload PDF, JPG, atau PNG

Pilih satu atau beberapa file. Bisa campur PDF dan gambar sekaligus. Seret untuk mengatur urutan.

2

Atur Opsi

Pilih ukuran kertas, gutter lipatan, dan arah halaman sesuai kebutuhan.

3

Proses Booklet

Algoritma Saddle-Stitch otomatis menyusun ulang halaman agar urut saat dilipat.

4

Cetak atau Unduh

Cetak langsung dari browser dengan satu klik, atau unduh PDF untuk dicetak nanti.

Pertanyaan Umum
Saddle-Stitch adalah metode penjilidan buku lipat paling umum. Halaman-halaman disusun sehingga saat dicetak bolak-balik dan dilipat di tengah, urutannya menjadi benar dari halaman 1 hingga akhir. Biasa digunakan untuk brosur, majalah, menu restoran, dan komik kecil.
Cukup pilih file JPG, PNG, atau WEBP saat upload — bisa campur dengan file PDF sekaligus. Setiap gambar otomatis dikonversi menjadi satu halaman PDF dengan ukuran A4, lalu digabungkan bersama file lainnya sesuai urutan yang Anda atur. Seret file untuk mengubah urutan halaman.
Setelah booklet selesai dibuat, klik tombol "Cetak Langsung ke Printer" di panel hitam. Browser akan membuka dialog cetak. Pastikan printer Anda sudah terkoneksi. Aktifkan Duplex/Bolak-balik → Flip on Short Edge, sesuaikan ukuran kertas, lalu klik Print/Cetak.
Setiap lembar kertas memiliki 4 permukaan halaman (depan-kiri, depan-kanan, belakang-kiri, belakang-kanan). Oleh karena itu total halaman harus kelipatan 4. Jika tidak, alat ini otomatis menambahkan halaman kosong di akhir dokumen.
100% aman. Seluruh proses berjalan di browser Anda menggunakan teknologi client-side (pdf-lib.js). File PDF maupun gambar Anda tidak pernah dikirim ke server manapun, sehingga privasi dokumen Anda sepenuhnya terjaga.
Gutter adalah ruang kosong tambahan di bagian tengah setiap lembar cetak, tepat di area lipatan. Ini berguna agar teks atau gambar yang berada di dekat tepi buku tidak tertutup saat dijilid. Untuk buku tipis (4–20 lembar), gutter 3–5 mm sudah cukup. Untuk buku lebih tebal, gunakan 8–15 mm.
